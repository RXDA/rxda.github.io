WR720Nv3折腾日记

WR720N OPENWRT完全教程（2017）

1.系统

使用我编译的带luci和usb ext4驱动的固件
链接: https://pan.baidu.com/s/1kUU4mpT 密码: 4nah

2.关闭路由器

格式化准备好的u盘为ext4，和swap,并插入路由器的usb口。
将路由器模式开关调为3g，开启路由器。
等到路由器灯不闪了，ssh登录路由器。

3.开始挂载

在luci界面挂载sda1为/overlay，挂载swap
复制系统文件到sda1

mount /dev/sda1 /mnt ; tar -C /overlay -cvf - . | tar -C /mnt -xf - ; umount /mnt

重启后再挂载一次

4.下载shadowsocks

https://github.com/shadowsocks/openwrt-shadowsocks/releases

先安装3个lib依赖，如有libp缺失，opkg update 后再安装

shadowsocks-luci

https://github.com/shadowsocks/luci-app-shadowsocks/releases

5.安装 chinadns

https://github.com/aa65535/openwrt-chinadns/releases

https://github.com/aa65535/openwrt-dist-luci/releases

6.更新CHinaDns

新建一个文件 /root/update_ignore_list.sh 写入如下内容: 

#!/bin/sh

set -e -o pipefail

wget -O- 'http://ftp.apnic.net/apnic/stats/apnic/delegated-apnic-latest' | \
awk -F\| '/CN\|ipv4/ { printf("%s/%d\n", $4, 32-log($5)/log(2)) }' > \
/tmp/chinadns_chnroute.txt

mv /tmp/chinadns_chnroute.txt /etc/

if pidof ss-redir>/dev/null; then
/etc/init.d/shadowsocks restart
fi

chmod +x /root/update_ignore_list.sh

打开路由器管.理页面 系统 - 计划任务 填写如下内容（每天 04:30 执行）: 

30    4     *     *     *     /root/update_ignore_list>/dev/null 2>&1
